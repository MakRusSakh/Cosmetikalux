## ЭТАП 1: Модальное окно (Quick View)

**Цель:** стимулировать добавление в корзину НЕ уходя из каталога.
**Триггер:** клик по карточке товара в каталоге.
**Время принятия решения:** 5–10 секунд.

### Десктоп: 960×540px, image LEFT (55%) + info RIGHT (45%)

### Элементы модалки (строгий порядок приоритета):

| # | Элемент | Зачем | Данные из CSV |
|---|---------|-------|---------------|
| 1 | Фото (2-4 шт, зум) | 93% покупателей решают по фото | photo_url |
| 2 | Бренд + название | Идентификация | brand, name |
| 3 | Рейтинг + кол-во отзывов | Социальное доказательство (+270% конверсия) | из БД |
| 4 | Цена + старая цена + скидка% | Ценность; неожиданная цена = #1 причина ухода | price |
| 5 | Цена за единицу (₽/мл, ₽/шт) | Сравнение выгоды | price / unit |
| 6 | Объём/вес | Понимание что получаешь | unit |
| 7 | Тип кожи (бейджи) | 52% хотят персонализацию | tags/метаданные |
| 8 | Шаг рутины | Уникально для K/J-beauty | метаданные |
| 9 | 3 ключевых ингредиента | Прозрачность = доверие | метаданные |
| 10 | CTA «В корзину» | Главное действие | — |
| 11 | «В избранное» | Сохранение для возврата | — |
| 12 | Ссылка «Подробнее» | Переход к полной странице | — |
| 13 | Флаг страны + бренд | Аутентичность (20% контрафакта на рынке) | country |
| 14 | Социальный триггер | FOMO: «Купили 234 раза» (реальные данные!) | из БД |

### Чего НЕТ в модалке (убивает конверсию):
- ❌ Полный состав INCI
- ❌ Длинное описание
- ❌ Отзывы (только счётчик)
- ❌ Доставка/возврат
- ❌ Похожие товары
- ❌ Табы и аккордеоны

### Информационная иерархия PDP (6 блоков):

| # | Блок | Что внутри | Почему здесь |
|---|------|-----------|-------------|
| 0 | Above the fold | Фото, название, цена, рейтинг, CTA, доставка | 60% решений — здесь |
| 1 | О товаре | Описание 2-3 предложения + бейджи (тип кожи, возраст, веган) | Быстрое понимание «для меня ли это» |
| 2 | Ключевые ингредиенты | 3-5 активных + plain-language описание + INCI (скрытый) | 52% покупателей ищут ингредиенты |
| 3 | Как использовать | Шаг рутины + визуальная цепочка + инструкция | Уникально для K/J-beauty, снижает барьер входа |
| 4 | Отзывы | Рейтинг-бар + фото покупателей + текстовые отзывы | Скролл до отзывов = +40% к покупке |
| 5 | Собери рутину | 2-4 комплементарных товара + бандл-скидка | +5.6% к среднему чеку |
| 6 | Характеристики | Таблица: объём, страна, тип кожи, возраст, текстура | Для дотошных покупателей |

## Психологические триггеры (встроены в оба формата)

| Триггер | Модалка | Страница | Механизм |
|---------|---------|----------|----------|
| Социальное доказательство | ★ рейтинг + «Купили 234 раза» | Полные отзывы + фото | 81% решают по отзывам |
| Ценовой якорь | Старая цена + % скидки | То же + цена за мл | Восприятие выгоды |
| Персонализация | Бейджи типа кожи | Бейджи + совет по типу кожи | 52% хотят «для меня» |
| Аутентичность | 🇰🇷 Корея + бренд | Оригинал + дистрибьютор | 20% контрафакта на рынке |
| Дефицит (реальный!) | — | «Осталось 3 шт» (если правда) | +30% скорость решения |
| Шаг рутины | «Шаг 7: Увлажнение» | Визуальная цепочка | Уникально для K-beauty |
| Бандл-мотивация | — | «3 средства = −10%» | +5.6% средний чек |

## Данные из CSV для карточки

Маппинг полей `products_tagged.csv` → элементы UI:

```
name         → Название товара
brand        → Бренд (над названием)
price        → Цена
unit         → Объём (рядом с ценой) + расчёт ₽/мл
photo_url    → Главное фото (+ доп. фото из медиабиблиотеки)
category     → Хлебные крошки
subcategory  → Хлебные крошки (уровень 2)
tags         → Бейджи (для_мужчин, для_детей, премиум)
country      → Флаг страны + «Оригинал»
```

Дополнительные поля (нужно добавить в БД):
- `ingredients[]` — ключевые ингредиенты (3-5)
- `inci_full` — полный состав INCI
- `skin_types[]` — типы кожи
- `routine_step` — шаг в корейской рутине (1-9)
- `age_group` — возрастная группа
- `how_to_use` — инструкция по применению
- `images[]` — массив фото (5-8 шт)
