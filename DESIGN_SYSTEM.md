# CosmetikaLux 2.0 — Дизайн-система

> Светлый премиальный стиль с корейским вайбом. Playfair Display как основной шрифт.
> Атмосфера: чистота, забота, роскошь, K-beauty эстетика.

---

## 1. Философия дизайна

### Ключевые принципы

| Принцип | Описание |
|---------|----------|
| **Чистота** | Много воздуха, минимум визуального шума, акцент на товаре |
| **Премиальность** | Элегантная типографика, изысканные акценты, благородные оттенки |
| **K-beauty вайб** | Мягкие градиенты, нежные тона, лепестки, стекло, росинки |
| **Доверие** | Аккуратная вёрстка, качественные фото, профессиональный тон |
| **Женственность** | Плавные формы, скруглённые углы, деликатные анимации |

### Настроение / Мудборд

Визуальные ассоциации:
- Утренняя роса на лепестке пиона
- Корейский чайный домик с мягким светом
- Флакон крема на мраморном столике
- Ханбок из шёлка пастельных тонов
- Нежный туман над горами на рассвете

## 8. Анимации и переходы

### Принцип

Все анимации — **деликатные и функциональные**. Никаких резких или агрессивных движений. K-beauty вайб = плавность и мягкость.

```
ПЕРЕХОДЫ (Transitions)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
--transition-fast:     150ms ease-out    (hover кнопок, иконок)
--transition-normal:   250ms ease-out    (раскрытие меню, модалки)
--transition-slow:     400ms ease-out    (появление секций, hero)

АНИМАЦИИ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
fade-in:         opacity 0→1, 400ms
slide-up:        translateY(20px→0) + opacity, 500ms
scale-in:        scale(0.95→1) + opacity, 300ms

Карточка hover:  translateY(-4px) + shadow-md, 250ms
Фото hover:      scale(1.05), 400ms, overflow:hidden
Кнопка hover:    brightness(1.05) + shadow, 150ms
Кнопка active:   scale(0.98), 100ms

Лепестки (hero): float animation, 8-12s, infinite, ease-in-out
Пульс (ИИ):     scale(1→1.1→1) + shadow-glow, 2s, infinite
Появление:       Intersection Observer → fade-in + slide-up при скролле
```
