## ПСИХОЛОГИЧЕСКАЯ МОДЕЛЬ ПОКУПАТЕЛЯ

### Путь от «зашла» до «купила» (beauty-специфика)

```
1. ЭМОЦИОНАЛЬНЫЙ КРЮЧОК (0-3 сек)
   «Ого, красиво. Это про уход за собой. Мне это нужно.»
   → Hero-баннер с aspirational-образом
   → Триггер: self-care identity, дофамин от новизны

2. ПРОВЕРКА ДОВЕРИЯ (3-10 сек)
   «А это настоящее? Не подделка? Можно ли доверять?»
   → Trust-бар: оригинал, доставка, возврат
   → Триггер: 20% подделок = страх → гарантия = облегчение

3. ОРИЕНТАЦИЯ (10-30 сек)
   «Что тут есть для моей кожи/проблемы?»
   → Категории ПО ПРОБЛЕМАМ (не по типу товара!)
   → Триггер: need-based shopping (российская специфика)

4. СОЦИАЛЬНОЕ ПОДТВЕРЖДЕНИЕ (30-60 сек)
   «А другие покупают? Им помогло?»
   → Бестселлеры с рейтингами + UGC-отзывы
   → Триггер: social proof, authority

5. ВОВЛЕЧЕНИЕ (1-3 мин)
   «О, вот это интересно! А ещё что есть?»
   → Новинки, бренды, ингредиентные спотлайты
   → Триггер: curiosity, FOMO, ingredient-savvy trend

6. КОНВЕРСИЯ (3-5 мин)
   «Добавлю в корзину. И вот это тоже.»
   → Карточки с «В корзину», промокод на первый заказ
   → Триггер: reciprocity (скидка), anchoring (старая цена)
```

### Три типа покупательниц CosmetikaLux

| Тип | Возраст | Что ищет | Что покажем на главной |
|-----|---------|----------|----------------------|
| **«Скинтеллектуал»** | 25-35 | Ингредиенты, составы, доказательства | Ингредиент-спотлайт, «клинически доказано» |
| **«Ритуалистка»** | 30-45 | Рутина, порядок, система ухода | «Подобрать рутину», наборы, шаги ухода |
| **«Результатница»** | 35-50 | Видимый результат, anti-age | До/после, отзывы, «результат за 28 дней» |

### СЕКЦИЯ 0: ANNOUNCEMENT BAR

**Психология:** Первое, что видит покупатель. Снимает барьер «а доставка дорогая?» до того, как он возникнет. Экономит «когнитивные ресурсы» для изучения товаров.

**Что показываем:**
```
🚚 Бесплатная доставка от 3 000₽  •  🎁 Скидка 10% на первый заказ: ПРИВЕТ10  •  📍 Южно-Сахалинск
```

**Поведение:**
- Горизонтальный скролл текста (если несколько сообщений)
- Можно закрыть (крестик справа)
- На мобильном: одно сообщение, ротация каждые 5 сек

**Дизайн-токены:**
```
Фон:         --accent-lavender (#C8A2C8)
Текст:       --text-inverse (#FFFFFF)
Шрифт:       Playfair 500, 13px, ls: 0.02em
Высота:      36px (десктоп) / 32px (мобайл)
Иконки:      Lucide, 14px, white
```

**Обоснование:** 89% top beauty e-commerce сайтов используют announcement bar. Бесплатная доставка с порогом увеличивает средний чек на 15-20% (покупатели «добирают» до бесплатной доставки).

### СЕКЦИЯ 1: HEADER

**Психология:** 30% посетителей сразу используют поиск. Поисковая строка должна быть визуально заметной. Корзина с бейджем — постоянное напоминание о незавершённой покупке.

**Структура (десктоп):**
```
┌────────────────────────────────────────────────────────────────┐
│                                                                │
│  CosmetikaLux    КАТАЛОГ  БРЕНДЫ  НОВИНКИ  АКЦИИ    🔍  ♡  🛒 │
│                                                          (2)   │
└────────────────────────────────────────────────────────────────┘
```

**Структура (мобайл):**
```
┌──────────────────────────────────┐
│  ☰   CosmetikaLux        🔍  🛒 │
│                             (2)  │
└──────────────────────────────────┘
```

**Ключевые решения:**

1. **Навигация «Каталог» вместо «Категории»** — привычнее для российского e-commerce
2. **«Новинки» в основном меню** — 68% покупателей импульсивны к новинкам в skincare
3. **«Акции» выделены цветом** — accent-rose для привлечения внимания
4. **Поиск — иконка → раскрывается в полную строку** — не загромождает хедер
5. **Счётчик на корзине** — bg: accent-lavender, text: white, минимум 20px

**Поведение при скролле:**
- Sticky top
- Glass-эффект: `rgba(255,255,255,0.92) + backdrop-filter: blur(12px)`
- Shadow: `--shadow-sm`
- Announcement bar скрывается при скролле вниз

**Mega-menu при наведении на «Каталог»:**
```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  ПО КАТЕГОРИЯМ         ПО ПРОБЛЕМЕ КОЖИ      ПОПУЛЯРНОЕ    │
│                                                             │
│  Кремы и эмульсии      Сухость и стянутость   🏆 Топ-10    │
│  Сыворотки             Акне и воспаления      🔥 Бестселлер │
│  Тонеры и эссенции     Пигментация            ✨ Новинка    │
│  Маски                 Морщины и anti-age                   │
│  Очищение              Расширенные поры        [Фото]       │
│  SPF-защита            Тусклый цвет лица       Sulwhasoo   │
│  Макияж                Чувствительность         Essential   │
│  Уход за телом                                  Cream       │
│  Наборы                                         4 500 ₽     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**Обоснование двойной навигации (категории + проблемы):**
Исследования показывают, что российские покупатели ищут «по проблеме» (need-based), а не по бренду. Двойная навигация обслуживает и тех, кто знает что ищет, и тех, кто ищет решение.

**Дизайн-токены:**
```

Hover:          --accent-primary + underline animation
Mega-menu:      --bg-surface, --shadow-lg, --radius-lg
```

**Ключевые решения:**

1. **Заголовок «Секреты безупречной кожи»** — не «Купите косметику», а outcome-based. Покупают не крем, покупают результат.
2. **«Оригинальная корейская»** — сразу отстройка от подделок (20% рынка).
3. **«С доставкой по всей России»** — снимает географический барьер для Сахалина.
4. **Два CTA:**
   - Primary: «Перейти в каталог» — для тех, кто знает что ищет
   - Secondary: «Подобрать уход для вашего типа кожи» — для тех, кто не знает (50%+ beauty-покупателей)
5. **Не карусель, а один сильный слайд** — карусели конвертят хуже статичного hero (данные Baymard). Но допустим 2-3 слайда с автопрокруткой каждые 7 сек.
6. **Фото: реальная девушка, не модель из стока** — 67% скептичны к «глянцевым» фото. Мягкий свет, натуральная красота, видна текстура кожи.

**Дизайн-токены:**
```

Фото:           object-fit: cover, radius-lg на десктопе
```

**Анимация входа:**
- Заголовок: fade-in + slide-up, 500ms, delay 200ms
- Подзаголовок: fade-in + slide-up, 500ms, delay 400ms
- CTA: fade-in + slide-up, 500ms, delay 600ms
- Фото: fade-in + scale(0.95→1), 600ms, delay 300ms
- Лепестки: float animation, infinite, 8-12s

### СЕКЦИЯ 3: TRUST BAR

**Психология:** Сразу после эмоционального крючка (hero) покупатель переходит в «режим проверки». «Красиво, но можно ли доверять?» Trust-бар отвечает на этот вопрос за 2-3 секунды. **+42% конверсии** — самый высокий ROI на единицу контента.

**Для России критически важно:** Гарантия подлинности (20% подделок), способы оплаты (Mir/СБП), скорость доставки.

**Макет:**
```
┌────────────────────────────────────────────────────────────────────┐
│                                                                    │
│  bg: --bg-secondary (#F7F3EF)                                      │
│                                                                    │
│  ✓ ОРИГИНАЛ         🚚 ДОСТАВКА          ↩ ВОЗВРАТ        🔒 ОПЛАТА │
│  100%               по всей              14 дней          Безопасная│
│  подлинность        России за            без вопросов     Mir / СБП │
│                     2-5 дней                                       │
│                                                                    │
└────────────────────────────────────────────────────────────────────┘
```

**Мобайл: горизонтальный скролл (swipeable) или 2x2 сетка.**

**4 USP — порядок обоснован:**

| # | USP | Почему именно этот | Психологический триггер |
|---|-----|--------------------|------------------------|
| 1 | 100% оригинал | Главный страх (20% подделок) | Снятие тревоги |
| 2 | Доставка по России | Сахалин → вся Россия | Доступность |
| 3 | Возврат 14 дней | «А если не подойдёт?» | Снятие риска |
| 4 | Безопасная оплата | Mir/СБП — привычные способы | Доверие к платежу |

**Дизайн-токены:**
```
Фон:          --bg-secondary (#F7F3EF)
Иконки:       Lucide, 28px, --accent-primary (#C8A2C8)
Заголовок:    Playfair 600, 15px, --text-primary, uppercase, ls: 0.04em
Описание:     Playfair 400, 13px, --text-tertiary
Padding:      --space-7 (48px) vertical
Разделители:  1px solid --border-light (десктоп) / нет (мобайл)
```

**Анимация:** fade-in при скролле (Intersection Observer), каждый элемент с delay 100ms.

### СЕКЦИЯ 4: ПОДБОР ПО ПРОБЛЕМЕ КОЖИ

**Психология:** Это КЛЮЧЕВАЯ секция для нашей ЦА. Российские покупатели ищут «по проблеме», не по бренду. 50%+ не знают, что именно им нужно. Эта секция отвечает: «Мы понимаем вашу проблему — вот решение».

**Почему не «Категории» (кремы, сыворотки)?** Потому что покупатель думает: «У меня сухая кожа» — а не «Мне нужна эмульсия». Категории по типу товара — это язык магазина. Проблемы кожи — язык покупателя.

**Макет (десктоп):**
```

│              Подберём уход под вашу задачу                          │
│              ─────── (золотая линия 40px) ───────                  │
```

- Не абстрактные иконки, а **close-up фото текстуры кожи или продукта**
- Мягкий gradient overlay снизу для текста
- Hover: scale(1.03) + shadow-md
- Каждая карточка — ссылка на `/catalog?concern=...`

| Hero | Секреты безупречной кожи | Outcome-based, aspirational |

| Проблемы | Подберём уход под вашу задачу | «Вашу» — персонализация |
| Бестселлеры | Хиты продаж | Просто и ясно |
| Промо | Весенняя забота о коже | Сезонная + эмоция |
| Новинки | Только что из Кореи | Свежесть + происхождение |
| Ингредиенты | Ингредиент месяца | Экспертность + регулярность |
| Отзывы | Что говорят наши покупатели | Social proof |
| Бренды | Наши бренды | Authority |
| Подписка | Будьте в курсе K-beauty трендов | FOMO + community |

## РЕЗЮМЕ: ПОЧЕМУ ЭТА СТРУКТУРА РАБОТАЕТ

```
ЭМОЦИЯ (Hero)
    ↓
ДОВЕРИЕ (Trust bar)
    ↓
НАВИГАЦИЯ ПО ПРОБЛЕМЕ (Подбор по коже)     ← «для тех, кто не знает»
    ↓
SOCIAL PROOF (Бестселлеры)                  ← «другие покупают»
    ↓
ИМПУЛЬС (Промо + FOMO)                     ← «надо брать сейчас»
    ↓
НОВИЗНА (Новинки)                           ← «дофамин от нового»
    ↓
ЭКСПЕРТНОСТЬ (Ингредиенты)                  ← «они разбираются»
    ↓
ПОДТВЕРЖДЕНИЕ (Отзывы/UGC)                  ← «реальные люди подтверждают»
    ↓
AUTHORITY (Бренды)                          ← «работают с серьёзными брендами»
    ↓
ЛОЯЛЬНОСТЬ (Подписка)                       ← «останусь на связи»
```

Каждая секция отвечает на конкретный вопрос покупателя в конкретный момент его пути. Порядок не случаен — он проведён от эмоции к рациональному решению, от «ого, красиво» к «добавить в корзину».
