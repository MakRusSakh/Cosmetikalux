## ЧАСТЬ 3. АУДИТ UX/UI ДИЗАЙНА

### 3.1. Визуальный дизайн

#### Цветовая схема

| Элемент | Цвет | HEX |
|---|---|---|
| Фон сайта | Чёрный | `#000000` |
| Основной текст | Белый | `#FFFFFF` |
| Заголовки | Белый | `#FFFFFF` |
| Фон кнопок | Чёрный | `#000000` |
| Акцент при наведении | Светло-фиолетовый | `#d186ff` |
| Активное состояние | Ярко-фиолетовый | `#db49ff` |
| Фон заголовка | Прозрачный | `transparent` |

**Оценка:** Чёрно-белая палитра с фиолетовыми акцентами создаёт контрастное, стильное впечатление, ассоциируется с «премиум». Однако для e-commerce это спорное решение:

| Плюсы | Минусы |
|---|---|
| Высокий контраст текста | Утомляет при длительном просмотре |
| Ощущение «премиальности» | Снижает доверие для некоторых ЦА |
| Выделяется среди конкурентов | Фото товаров теряются на тёмном фоне |
| | Не типично для сегмента beauty/косметика |

#### Типографика

| Шрифт | Назначение | Оценка |
|---|---|---|
| **Oswald** | Заголовки (FontHeader) | ✅ Хорошо читается, жирный, современный |
| **Roboto** | Основной текст (FontParagraph) | ✅ Хорошая читаемость |

⚠️ Загружаются избыточные наборы символов (кириллица, латиница, греческий, вьетнамский) — достаточно только кириллицы и латиницы.

#### Визуальная иерархия

| Экран | Содержимое | Оценка |
|---|---|---|
| **1-й экран (above fold)** | Шапка + промо-баннер «Подарочные сертификаты 🎁» | ⚠️ Не хватает hero-изображения |
| **2-й экран** | Блок «Новинки» (4 товара Medicube) | ОК |
| **3-й экран** | Каталог (6 категорий) + контакты | ОК |

**Проблема:** Первый экран не содержит убедительного визуального хука — нет hero-баннера с фото продукции. Текстовый промо «Подарочные сертификаты» — слабый первый контакт для нового посетителя.
