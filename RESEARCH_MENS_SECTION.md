## 1. Психология мужских покупок в health/beauty

### Как мужчины принимают решение о покупке

- **Целевые покупатели**: пришёл → нашёл → купил. Меньше исследований, чем у женщин
- 60% мужчин используют 1-2 средства (vs 46% женщин с 3+ средствами)
- Приоритет: функциональность → цена → бренд (женщины: рекомендации → холистический подход)
- Решение быстрее, но уходят при первой же путанице
- «Satisficers» (удовлетворяющиеся) — дайте чёткий «лучший вариант» и они купят

### ТОП-5 факторов доверия для мужчин

| # | Фактор | Влияние на конверсию |
|---|--------|---------------------|
| 1 | Рейтинг + реальные отзывы | +380% |
| 2 | Безопасность сайта (SSL, способы оплаты) | 46% уходят при сомнениях |
| 3 | Сертификаты качества (GMP, JHFA) | Критично для японских БАДов |
| 4 | Понятная политика возврата | 91% считают важным |
| 5 | Аутентичность бренда (не инфлюенсеры) | Продукт говорит сам за себя |

### Язык, который работает на мужчин

```
✅ Функциональный, результат-ориентированный:
   «Энергия и выносливость. Курс 30 дней. Результат с 2-й недели.»
   «Витамины для мужчин 30-39: цинк 15мг, B-комплекс, CoQ10»
   «83 ₽/день — дешевле чашки кофе»

❌ Эмоциональный, размытый:
   «Почувствуйте гармонию тела и духа»
   «Позаботьтесь о себе с любовью»
   «Ваш путь к внутренней красоте»
```

- Для новых клиентов: «уход ОТ проблемы» (от усталости, от стресса)
- Для повторных: «движение К результату» (больше энергии, лучше сон)

### Что заставляет мужчин уйти без покупки

| Причина ухода | % мужчин |
|---------------|----------|
| Скрытые расходы на доставку | 48% |
| Сомнения в безопасности сайта | 46% |
| Медленная доставка | 29% |
| Сложный чекаут / регистрация | 22% |
| Непонятно, подходит ли мне | ~30% (оценка) |

### Что триггерит «купить сейчас»

- Прозрачная цена «всё включено»
- Бесплатная доставка (70% мужчин — фактор #1)
- Гостевой чекаут (без регистрации)
- Рейтинг + отзывы на видном месте
- Понятный возврат

### Японские БАДы: что ценят мужчины

- Перформанс: энергия, выносливость, сила (не красота)
- Натуральные ингредиенты (японские законы запрещают гормоны → растительные/морские экстракты)
- Формулы по возрастам (FANCL Men's 20-29 / 30-39 / 40-49 / 50-59 / 60+)
- Качество: JHFA-марка, GMP-стандарт, японский контроль безопасности
- Наследие бренда: FANCL (с 1980), DHC (с 1972), Asahi Dear-Natura — доверие через историю

### Парадокс выбора

> «6 вариантов конвертят в 10 раз лучше, чем 24» — The Jam Study

- Мужчины хотят **курируемую простоту** + детали по запросу
- Показывать 5-8 hero-товаров, не весь каталог
- Блок «Рекомендуем» с 1 товаром-бестселлером на каждую задачу
- Подробности — по клику, не по умолчанию

## 2. Цветовая психология для мужской аудитории

### Топ-3 цвета доверия у мужчин (Joe Hallock Study)

1. **Синий** — #1 любимый цвет мужчин. Navy = доверие + премиальность
2. **Зелёный** — #2. Здоровье, витальность, рост
3. **Тёмно-серый** — #3. Мужчины предпочитают shades (цвет + чёрный), а не tints (цвет + белый)

> 62-90% потребителей оценивают товар по цвету (Kissmetrics)

### Что работает: мягкая маскулинность

| Подход | Примеры | Результат |
|--------|---------|-----------|
| Muted navy + sage | Hims, FANCL Men's | Премиум + спокойствие |
| Charcoal + white + тёплый акцент | Hims, Roman | Современность + чистота |
| Warm grey фон | DHC, iHerb | Не стерильно, не агрессивно |
| Solid teal/green CTA | Roman, iHerb | Доверие + действие |

### Что НЕ работает

| Цвет | Почему нет |
|------|-----------|
| Розовый / пурпурный | Читается как женский для health-товаров |
| Ярко-оранжевый / коричневый | Наименее предпочитаемые мужчинами |
| Неон | Дешевизна, подрывает доверие |
| Чистый чёрный #000000 | Слишком агрессивный → charcoal #2D3436 |
| Стерильный белый (один) | Холодно → добавить тёплые акценты |

### Референсные бренды

**Hims (forhims.com):**
- Глубокий серый #333333 + белый + тёплые акценты (персик/коралл)
- «Glossier для мужчин» — мягкий, muted, lifestyle
- Sans-serif типографика, много воздуха

**Roman (ro.com):**
- Белый + navy + красный акцент
- Медицинская эстетика: врачи, клинические данные
- Как медицинский портал, а не магазин

**FANCL Men's (official):**
- Белый/серый фон + тёмные акценты (navy, charcoal, dark green)
- Структурированная сетка, информационно-насыщенная
- Наследие бренда как главный trust-сигнал

**iHerb (men's section):**
- Белый фон, плотная сетка 4-5 колонок
- Зелёные CTA, звёзды + отзывы как главный trust
- Утилитарный дизайн → конверсия

### Общий паттерн всех успешных брендов

> Маскулинность через **вычитание** (убираем пастель, курсив, декор),
> а НЕ через **добавление** (без черепов, огня, экстрима).
> Результат: гендерно-нейтральный дизайн с лёгким уклоном в клинику/структуру.

## 3. Итоговая палитра для CosmetikaLux «Для мужчин»

### Основная палитра

| Роль | Название | HEX | Применение |
|------|----------|-----|------------|
| Primary | Muted Navy | `#34435E` | Hero-фон, заголовки секций |
| Secondary | Slate | `#6D8196` | Подзаголовки, иконки, бордеры |
| Accent | Sage Green | `#9CAF88` | Бейджи здоровья, подзаголовок hero |
| Background | Warm White | `#FAF9F6` | Основной фон страницы |
| Surface | Light Warm Grey | `#F0EDEA` | Карточки товаров, блок доверия |
| Text | Charcoal | `#2D3436` | Основной текст |
| Text Muted | Medium Grey | `#636E72` | Описания, подсказки |
| CTA | Deep Teal | `#0D9488` | Кнопка «В корзину» |
| CTA Hover | Darker Teal | `#0F766E` | Hover-состояние |
| Sale/Urgency | Muted Red | `#C53030` | Скидка%, «Осталось 3 шт» |

### Сравнение с основным каталогом CosmetikaLux

| Токен | Основной каталог | Мужская секция | Почему |
|-------|-----------------|----------------|--------|
| `--bg` | `#FDFBF9` тёплый | `#FAF9F6` чуть холоднее | Менее «уютный», более чистый |
| `--surface` | `#FFFFFF` белый | `#F0EDEA` warm grey | Структура + маскулинность |
| `--accent` | `#C8A2C8` lavender | `#9CAF88` sage | Здоровье вместо красоты |
| `--cta` | gradient lavender→rose | solid `#0D9488` teal | Доверие + действие |
| `--hero-bg` | gradient pastel | `#34435E` navy | Авторитет без агрессии |
| `--heading` | 600 weight | 700 weight | Чуть увереннее |

### Типографика: Playfair Display — единственный шрифт

> **ПРАВИЛО:** В мужской секции используется ТОЛЬКО Playfair Display.
> В основном каталоге: Playfair (заголовки) + Inter (body).
> В мужской секции: Playfair Display ВЕЗДЕ — заголовки, body, кнопки, цены, бейджи, формы.

Это создаёт монолитную премиальную типографику — единый голос секции.
Playfair Display хорошо читается и в body-размерах (16px+), особенно weight 400.

| Элемент | Вес | Размер |
|---------|-----|--------|
| Hero h1 | 700 | 48px/56px |
| Hero subtitle | 400 | 18px/28px |
| Заголовки секций h2 | 700 | 32px/40px |
| Подзаголовки h3 | 600 | 24px/32px |
| Body текст | 400 | 16px/26px |
| Описания, подсказки | 400 | 14px/22px |
| Цена | 700 | 28px/36px |
| Цена за день | 400 | 13px/20px |
| Кнопка CTA | 600 | 15px/20px, tracking 0.04em |
| Бейджи | 600 | 11px/16px, tracking 0.08em |
| Рейтинг/отзывы | 400 | 14px/20px |

### Гармония палитр

Мужская палитра гармонирует с основной через:
- **Тёплый белый** — общая база (`#FAF9F6` ≈ `#FDFBF9`)
- **Playfair Display** — тот же шрифт, но единственный (без Inter)
- **Структура** — те же компоненты (ProductCard, Badge, PriceDisplay)
- **Переход** — при переходе из каталога в мужскую секцию нет «ломки»

### CSS-переменные

```css
.men-section {
  /* Типографика: единый шрифт */
  --men-font: 'Playfair Display', Georgia, 'Times New Roman', serif;

  /* Цвета */
  --men-primary: #34435E;
  --men-secondary: #6D8196;
  --men-accent: #9CAF88;
  --men-bg: #FAF9F6;
  --men-surface: #F0EDEA;
  --men-text: #2D3436;
  --men-text-muted: #636E72;
  --men-cta: #0D9488;
  --men-cta-hover: #0F766E;
  --men-sale: #C53030;

  /* Hero gradient */
  --men-hero-gradient: linear-gradient(135deg, #34435E 0%, #2D3436 60%, #1A2332 100%);
}

/* Все элементы внутри мужской секции — только Playfair Display */
.men-section,
.men-section * {
  font-family: var(--men-font);
}
```
