## ЧАСТЬ 9. ВЫЯВЛЕННЫЕ ПРОБЛЕМЫ (ПРЕДВАРИТЕЛЬНЫЕ)

На основе первичного анализа уже выявлены следующие проблемы:

### Критические

1. **Отсутствует sitemap.xml** — поисковые системы не имеют карты сайта для индексации
2. **URL-структура не семантична** — используются числовые ID категорий (/products/category/5099886) вместо читаемых URL
3. **Ограничения платформы Nethouse** — ограниченные возможности кастомизации, SEO, функциональности

### Важные

4. **Страница доставки использует системный URL** (/page/1484327) — не SEO-friendly
5. **Чёрный фон сайта** — потенциально снижает читаемость и воспринимаемый уровень доверия для e-commerce
6. **25+ категорий в одноуровневом списке** — нет иерархической структуры, перегружает навигацию
7. **Нет sitemap в robots.txt** — даже если sitemap существует, он не указан в robots.txt

### Требуют проверки

8. Отсутствие или наличие микроразметки
9. Состояние мета-данных (title, description)
10. Скорость загрузки и Core Web Vitals
11. Правовое соответствие (152-ФЗ, оферта)
